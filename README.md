# Intervue

## Description
Intervue is a platform designed to help users prepare for interviews through mock interviews and resources tailored to various industries and roles.

## Technology Stack
- **Frontend:** React, CSS, HTML
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Deployment:** Heroku

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/AtharvChanana/Intervue.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Intervue
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
